# Smart Math

Multi-purpose mathematical toolbox built with Python + PyQt5.

## Features

| Category | Tools |
|----------|-------|
| **Algebra** | Calculator, Power, Square Root, Quadratic Solver |
| **Number Theory** | Prime Checker, Prime Factorization, GCD & LCM, Factorial |
| **Geometry** | Circle Calculator, Polygon Angle, Pythagoras Theorem |
| **Trig & Log** | Trigonometry (sin/cos/tan + inverses), Logarithms (ln/log₁₀/arbitrary base) |
| **Combinatorics** | Combinations (nCr), Permutations (nPr) |
| **Statistics & Conversion** | Mean/Median/Mode/Std Dev/Variance, Base Converter (bin/oct/dec/hex), Percentage |

## Requirements

- Python 3.8+
- PyQt5

```bash
pip install PyQt5
```

## Run

```bash
python script.py
```
